部署过程中遇到的一些问题

comware 平台

没有找到匹配字符，无法连接到目标主机。

查看 log 错误提示如下：

错误原因：

  • 交换机主机名中含有特殊字符:，但是comware.rb模型中正则表达式没有匹配:

解决方法：

  • 编辑comware.rb`配置文件：匹配主机名特殊字符的正则表达式。

登陆使用telnel协议的设备失败，使用ssh协议的设备正常。

错误原因：

  • 因为华三交换机comwre平台版本不同，telnet登陆提示符也有差异。所以comware.rb模型配置中，匹配telnet登陆提示的正则表达式没有匹配所有版本的登陆提示符。

解决方法：

  • 编辑comware.rb配置文件：修改匹配telnet登陆提示符的正则表达式。

    • 华三 comware v7 平台登陆提示符 login:后面有个空格写正则表达式的时候注意匹配到！
